## 房源说明

### 网络 / Wi-Fi

- **网络名称:** Monkeymousewifi
- **密码:** bendalong
- **Wi-Fi 信号增强器**位于楼上 — 如果网速慢,请将设备的 Wi-Fi 关闭再打开,设备会自动连接最强信号。
- 备注:南阿尔卑斯湖区由于天气条件和高峰期需求,Wi-Fi 偶尔会出现速度变慢的情况。

### 取暖

- **地暖**已设定在 **20°C**,请勿调节控制器(系统需要约 2 天才能改变温度)。
- 热泵和燃气壁炉的操作说明请参见下方**电器**部分。

### 花园 + 步道

- 花园里有水果树(榅桲、杏、苹果) — 欢迎自取。
- 通过后花园可直接进入**皇后镇山步道(Queenstown Hill walking track)** — 推荐从房屋出发散步。
- 请在夜间将**户外坐垫收回室内**,以免次日潮湿。

### 其他设施

- **自清洁烤箱** — 请勿使用化学清洁剂;按屏幕提示进行自清洁。
- **洗碗机洗涤剂**:只使用**所提供的洗碗粉** — 使用液体会导致故障,维修费用由住客承担。
- **餐桌**:请使用所提供的餐垫和杯垫 — 请勿将热物直接放在餐桌上。
