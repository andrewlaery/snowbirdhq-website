## 物件案内

### インターネット / Wi-Fi

- **ネットワーク名:** Monkeymousewifi
- **パスワード:** bendalong
- 上階に **Wi-Fi 中継器** があります。インターネットが遅い場合は Wi-Fi を一度オフにして再度オンにすると、お使いの端末が最も強い信号を選択します。
- 注意:サザンレイクス地域では、天候や繁忙期の利用集中により Wi-Fi が遅くなることがあります。

### 暖房

- **床暖房** は各部屋で **20°C** に設定されており、調整しないでください(温度変更には約 2 日かかります)。
- ヒートポンプとガス暖炉の操作方法は、下記の**家電**セクションをご覧ください。

### 庭 + ハイキングコース

- 庭には果樹(マルメロ・アプリコット・りんご)があります。ご自由にお取りください。
- 裏庭から **クイーンズタウン・ヒル・ウォーキングトラック** に直接アクセスできます。家から散歩に出かけるのに最適です。
- **屋外クッション**は、翌日湿らないよう、夜間は室内にお持ちください。

### その他の設備

- **セルフクリーニングオーブン** — 化学洗剤は使用しないでください。画面の指示に従ってセルフクリーニングを実行してください。
- **食器洗い機の洗剤**:**付属の食器洗い機用粉末**のみご使用ください。液体は故障の原因となり、修理費用が発生します。
- **ダイニングテーブル**:付属のランチョンマットとコースターをご使用ください。熱いものを直接置かないでください。
