## 房屋概览

### 网络 / Wi-Fi

Wi-Fi 名称及密码已在上方快速参考信息中列出。Wi-Fi 调制解调器放置在第二起居室的书桌上。

### 房屋布局

- 主生活楼层设有起居室（热泵供暖）、第二起居室（燃气壁炉）、完整厨房及餐厅。
- 3间卧室及配套卫浴。
- 车库可从屋内直接进入。

### 床位配置

- **3间卧室** — 1张大床（主卧）、1张标准双人床、1间上下铺房间配2张单人床（最多可住6位客人）。
- **婴儿床及高脚椅**可提前告知我们为您准备。

### 供暖

房屋提供多种供暖方式——具体操作请参阅"家电 → 供暖"部分：

- **三菱电机热泵**，位于主起居室。
- **Rinnai 燃气壁炉**，位于第二起居室。
- **便携式电暖器**，置于每间卧室衣柜内。
- **毛巾加热架**，位于各卫浴间——请在退房前关闭。

### 热水

- **400升热水箱** — 满箱可供约7次淋浴使用。
- **再加热时间：** 约需1小时——如有多位客人，建议错开淋浴时间。

### 自来水

房屋内所有自来水均可直接饮用。

### 车库

- 可通过内部门直接从屋内进入车库。
- **车库门：** 车库内设有墙壁开关。若车库门关闭后自动重新开启，请再次按下按钮——偶尔可能出现此情况。
- 车库内设有洗衣房、清洁用品，并提供**滑雪装备存放**空间。
- 入口附近备有**大型垃圾桶**，可供室内晾干滑雪靴使用。

### 室外区域

- 花园内提供BBQ烧烤炉（详见"家电 → 室外"部分）。
- 花园区域开放时间为每日早上7点至晚上10点。

### 清洁用品

清洁用品放置在车库水槽下方（橙色收纳桶——内含疏通剂）。

### 公共设施

如遇停电，请访问 [Aurora Energy 皇后镇停电信息页面](https://www.aurora.energy/outages) 了解区域供电情况。

### 出行方式

- **公交：** 从 Jones Avenue 乘坐5路公交车（每小时一班，运营时间6:05am–10:05pm）。如需实时时刻表，请使用 Transit 应用程序。
- **Uber：** 皇后镇可使用 Uber 叫车服务。
- **出租车：** QT Taxis 03 450 3000。
- **皇后镇市中心：** 驾车约数分钟可达。
- **阿罗镇：** 驾车约10分钟。

### 附近购物

- **Night n Day Kawarau** — 23 Eleventh Avenue，步行约5分钟（便利店，营业至深夜）。
- **Pak'n'Save / Countdown** — 弗兰克顿，驾车约5分钟。
- **Four Square** — 市中心。

### 附近餐厅

- **11th Avenue by Franks** — 步行约5分钟（03 222 7030）。
- **The Hayes** — Lake Hayes Estate，步行约15分钟（03 441 4074）。

### 徒步与活动

**Twin Rivers Trail** — 全程约2至3小时的环形步道，途经 Shotover Bridge，可从附近区域直接出发。

### 住宿中期保洁

如有需要，可额外付费安排中期保洁服务——请联系房东进行预约。

### 退房须知

- 将钥匙（房屋钥匙**及**车库门禁卡）放回密码锁箱。
- 将餐具放入洗碗机并启动清洗。
- 将垃圾分别投入红色/蓝色/黄色垃圾桶。
- 关闭所有灯具、毛巾加热架、燃气壁炉及BBQ烧烤炉。
- 床上用品请保留原位，毛巾请放回各自所在的房间。
