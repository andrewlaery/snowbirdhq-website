### Bosch Combination Oven/Microwave (HBC86Q650A/02)

Built-in **combination oven + microwave** unit, 2450W / 2900W — kitchen built-in.

#### Oven Mode
1. **Ensure the wall power switch labelled "oven" is turned ON first.** This is the most common cause of "oven won't turn on".
2. Press the power button on the unit.
3. Select cooking function with the control dial.
4. Set temperature and timer.
5. Press **Start**.

#### Microwave Mode
1. Place food in a microwave-safe dish.
2. Select microwave function.
3. Set power level + time.
4. Press **Start**.
