### Electric Oven

Standard built-in electric oven below the gas cooktop.

- **Operating**: select cooking function (fan/convection, grill, conventional bake), set temperature, allow ~10 min to preheat to 180°C.
- **Timer**: set on the unit if needed; oven auto-shuts off when timer expires.
- Use the supplied trays + oven gloves (in the second drawer).
- **Switch off at the dial** when finished.
