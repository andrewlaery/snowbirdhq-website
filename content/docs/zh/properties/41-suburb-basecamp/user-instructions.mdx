## 房源导览

### 网络 / Wi-Fi

网络名称和密码已在上方快速参考中列出。

### 冷暖空调

本房源配备一台 **Daikin FVXG50K2V1B Nexura** 落地式冷暖两用空调（约 5.0 kW，制热 + 制冷），前置辐射热面板可快速供暖，且无吹风感。建议您先通过增减衣物来调节体感温度；如有任何异常，请联系房东。详细操作说明请参阅"家电 → 暖气"中的 Daikin Nexura 条目。

天气较凉时外出，请关闭所有门窗，以便回来时室内仍保持舒适温度。

### 厨房

房源配备设施齐全的精品厨房，包括：Miele 洗碗机（AutoDos 自动投放）、Beko 电烤箱、Bosch 电磁炉、Beko 嵌入式微波炉及烧烤功能、Beko 对开门冰箱/冷冻柜（带外置饮水及制冰功能）、电热水壶、搅拌机、烤盘及完整烹饪器具。烹饪基础用品（食用油、盐、胡椒、茶、咖啡）均已备好。**自来水可直接饮用。** 各设备的操作说明请参阅"家电 → 厨房"部分。

### 卫生间

共三间卫生间，每间均备有毛巾、洗发水、护发素、沐浴露及洗手液。主卧套间内配有吹风机。主卫设有电热毛巾架——退房时请记得关闭。

### 洗衣房

室内配备 Miele 洗衣机及热泵烘干机，另提供晾衣架、熨斗及烫衣板。操作说明请参阅"家电 → 洗衣"部分。

### 户外区域

私人露台可俯瞰瓦卡蒂普湖（Lake Wakatipu）及非凡峰（The Remarkables）。花园修缮整洁，草坪与柏树相映成趣。户外用餐家具及燃气烧烤炉一应俱全——烧烤炉操作说明请参阅"家电 → 户外"部分。

### 公共设施 / 停电处理

如遇停电，可访问 [Aurora Energy 皇后镇停电信息页面](https://www.aurora.energy/outages) 查询区域供电状态。

### 出行交通

驾车前往皇后镇市中心约需 5–7 分钟。山脚处设有公共巴士站，班次频繁，具体路线请查阅 Orbus 皇后镇时刻表。步行悠闲前往市区约需 15–20 分钟。

### 垃圾与回收

- 红色垃圾桶（一般垃圾）：每周收取一次。
- 蓝色垃圾桶（玻璃）与黄色垃圾桶（混合可回收物）隔周交替收取。
- 具体时间表请访问 [QLDC 垃圾及回收收集页面](https://www.qldc.govt.nz/services/rubbish-recycling/rubbish-recycling-collection/)，或参考邻居当天摆放的垃圾桶颜色。
- 请在收取当日早上 7 时前或前一晚将垃圾桶推至路边，并在收取当日傍晚前将垃圾桶收回。

### 儿童设施

如有需要，可提供婴儿旅行床、儿童餐椅及少量玩具——请在抵达前提前告知房东，以便提前备好。
