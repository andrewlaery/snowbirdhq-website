### Bosch Microwave Combination Oven (HBC86Q650A/02) — Top Unit

The kitchen has **two stacked Bosch ovens**. This is the **TOP** unit — a "speed oven" combining microwave + convection/grill in one cavity (~44L, 1980W oven / 2250W grill / 900W microwave).

- **Use this for**: fast cooking, reheating, microwaving, or speed-roasting smaller items.
- For full-size roasts and large trays, use the **bottom unit** — the [Bosch single wall oven](#) (HBN560550A/02).

#### Oven Mode (Convection / Grill / Combination)
1. **Ensure the wall power switch labelled "oven" is turned ON first** — both ovens share this isolator. If the oven won't turn on, check this switch.
2. Press the power button on the unit.
3. Select cooking function with the control dial.
4. Set temperature and timer.
5. Press **Start**.

#### Microwave Mode
1. Place food in a **microwave-safe dish**.
2. Select microwave function.
3. Set power level — buttons are labelled **90 / 180 / 360 / 600 / 900** (these are watts, not %).
4. Set time and press **Start**.

#### MicroCombi (Microwave + Grill Together)
- Combines microwave + grill for **fast cooking with browning** — e.g. baked potatoes, gratins, frozen pies.
- Select MicroCombi mode, choose microwave wattage + grill level, set time.

#### Autopilot (Pre-set Programs)
- ~15 pre-set programs for common dishes (frozen pizza, rice, fish, etc.).
- Select Autopilot, choose the program number, enter weight when prompted, press **Start**.
