## 房屋介绍

### 网络 / Wi-Fi

Wi-Fi 网络名称已在上方快速参考信息中列出。密码请参见入住时提供的欢迎卡片。

### 供暖

皇后镇地处新西兰南阿尔卑斯山脉，气候变化多端——有时白天气温超过 30°C，隔天便可能接近冰点。将室内温度调节至理想状态需要一点耐心。

本房屋（Wright's Garden）配备了 **Daikin 中央供暖系统**，可为全屋提供均匀稳定的温度。系统已预设至适合大多数住客的温度。部分房间支持单独调节温度——如有任何不适，请随时联系房东。

寒冷天气下，请保持门窗关闭以留住热量。如感到不适，建议先适当增减衣物；若室温仍不理想，再联系房东。

在凉爽天气外出时，请确保关好所有门窗，以便您回来时室内依然温暖舒适。

两间浴室及洗衣房均铺设了**地暖**。每个房间墙上均设有 LED 面板开关用于控制地暖——按下面板即可开启。

浴室内的**暖毛巾架**由独立开关控制，该开关位于毛巾架旁边的墙上（而非 LED 面板）。退房时，请务必关闭地暖及暖毛巾架。

此外，房屋还备有 **3 台便携式插电暖气**，可在需要时即时提供额外暖意——如需使用，请向房东询问存放位置。

### 照明

所有灯具均由附近墙上的开关控制。部分房间（包括卧室）配有调光开关——若灯光出现闪烁，请按住调光开关直至亮度调至最高，闪烁即可消除。

### 儿童设施

- **玩具：** 各类儿童玩具存放在客厅的橱柜中。
- **餐具：** 厨房备有一套儿童专用餐具，存放于厨房抽屉内。
