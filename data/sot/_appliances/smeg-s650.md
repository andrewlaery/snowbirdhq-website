### Smeg Top-Loading Washing Machine (S650, 6.5kg)

Top-loading washing machine, **6.5 kg** capacity — laundry room.

1. Load laundry into the drum (max 6.5 kg).
2. Add detergent to the dispenser.
3. Select wash program — **Cotton** for everyday, **Delicate** for sensitive fabrics, **Quick wash** for lightly soiled.
4. Set water temperature if needed.
5. Press **Start**.
