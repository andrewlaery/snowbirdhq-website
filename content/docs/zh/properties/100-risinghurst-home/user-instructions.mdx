## 房屋导览

### 网络 / Wi-Fi

Wi-Fi 名称及密码已在上方快速参考中注明。Wi-Fi 调制解调器位于第二客厅的书桌上。

### 房屋布局

- 主生活层设有客厅（配热泵空调）、第二客厅（配燃气壁炉）、完整厨房及餐厅。
- 3间卧室及配套卫浴。
- 车库可从屋内直接进入。

### 床位配置

- **3间卧室** — 主卧1张大床（king size）、次卧1张大床（queen size）、高低床房间含2张单人床（最多可住6位宾客）。
- **婴儿床及宝宝餐椅**可提前告知房东预约使用。

### 供暖

房屋配备多种供暖方式，具体操作请参阅"家电 → 供暖"部分：

- **Mitsubishi Electric 热泵空调** — 位于主客厅。
- **Rinnai 燃气壁炉** — 位于第二客厅。
- **便携式电暖器** — 每间卧室衣柜内均有配置。
- **浴室电热毛巾架** — 请于退房前关闭。

### 热水

- **400升热水储水箱** — 满箱可供约7次淋浴使用。
- **加热时间：** 约需1小时 — 如有多人洗澡，建议错开时间。

### 饮用水

房屋内各处自来水均可直接饮用。

### 车库

- 可通过室内门直接进入车库。
- **车库门：** 车库内设有墙壁开关。如门关闭后自动重新开启，请再按一次按钮 — 偶尔会出现此情况。
- 车库内设有洗衣区、清洁用品，并提供**滑雪器材存放**空间。
- 入口附近备有**大型收纳箱**，可在室内晾干滑雪靴。

### 户外区域

- 花园内备有烧烤架（详见"家电 → 户外"部分）。
- 花园区域开放时间为每天早上7点至晚上10点。
- **请在晚上10点后保持安静** — 周围邻居距离较近，敬请谅解。

### 清洁用品

清洁用品存放于车库水槽下方（橙色收纳桶内，含通渠清洁剂）。

### 公用设施

如遇停电，可访问 [Aurora Energy 皇后镇停电信息页面](https://www.aurora.energy/outages) 查询区域供电情况。

### 出行方式

- **公共巴士：** 5路公共汽车，乘车点位于 Jones Avenue（每小时一班，运营时间6:05am–10:05pm）。可使用 Transit 应用程序查看实时时刻表。
- **Uber：** 皇后镇内可使用。
- **出租车：** QT Taxis 03 450 3000。
- **皇后镇市中心：** 驾车即达。
- **箭镇（Arrowtown）：** 驾车约10分钟。

### 附近购物

- **Night n Day Kawarau** — 23 Eleventh Avenue，步行约5分钟（便利店，营业至深夜）。
- **Pak'n'Save / Countdown** — 弗兰克顿，驾车约5分钟。
- **Four Square** — 市中心。

### 附近餐厅

- **11th Avenue by Franks** — 步行约5分钟（03 222 7030）。
- **The Hayes** — Lake Hayes Estate，步行约15分钟（03 441 4074）。

### 步行与户外活动

**Twin Rivers Trail** — 环形步道，全程约2至3小时，途经 Shotover Bridge，可从附近区域步行前往。

### 中途保洁

如需在入住期间安排保洁服务，需额外付费，请联系房东预约。

### 退房须知

- 将钥匙（房门钥匙**及**车库遥控钥匙）放回密码锁盒。
- 将碗碟放入洗碗机。
- 将垃圾分别投入红色/蓝色/黄色垃圾桶。
- 关闭所有灯具、浴室电热毛巾架、燃气壁炉及烧烤架。
- 床上用品请留在原处，毛巾请放回各自对应的房间。
