# User Instructions

### Internet / Wi-Fi

The Wi-Fi network name is shown above in the Quick Reference. Your password is provided on the welcome card at check-in.

### Access & Security

- **Entry:** The front door uses an electronic lock. Your access code is sent a few days before arrival in your check-in information.
- **Garage:** Press the button labelled "garage" on the light switch panel beside the front door.

### Heating

- Located in New Zealand's Southern Alps, Queenstown's climate can vary wildly — 30°C+ one day, near-freezing the next. Achieving the perfect temperature in the house can take some doing.
- Wright's Garden has a Daikin Central Heating System that provides even and consistent temperatures throughout. The system is pre-set at a level that suits most people.
- Several rooms allow temperature adjustments — let the host know if anything feels off.
- In cold weather, keep doors and windows closed to retain heat.
- For comfort: adjust clothing first, contact the host if the temperature is still wrong.

When leaving the house for periods of time in cooler weather, please ensure all doors and windows are closed so the house stays at a good temperature for your return.

### Lighting

All lights are controlled by wall switches near their location. Several rooms (including bedrooms) have dimmers — if lights flicker, press and hold the dimmer switch until it reaches full brightness to stop the flickering.
