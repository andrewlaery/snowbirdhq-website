# 重要情報

* **緊急の場合：**
    * 救急車・消防・警察が必要な際は、ニュージーランド国内の公衆電話・一般電話・携帯電話から **111** へお電話ください。

### 健康と安全

* 当ロッジには認定済みの煙感知器と、緊急時に使用する消火器が設置されています。
* **緊急出口：** 緊急事態が発生した場合は、正面玄関からロッジの外へ避難してください。緊急集合場所は、建物から安全な距離が確保された屋外となっております。

### 緊急時の対応手順 — 火災

* ロッジには煙感知器が設置されています。絶対に取り外さないようお願いいたします。
* **火災発生時の行動：**
    * 落ち着いて行動してください。
    * 寝室のドアに触れてください。熱い場合は、絶対に開けないでください。
    * 部屋から出られる場合は、最寄りの出口へ向かってください。
    * 必要に応じて床を這って進み、煙の下側を移動してください。
    * 建物から安全な距離が確保できる屋外へ集合してください。
    * **111** へ電話し、消防署へ通報してください。
    * 許可が下りるまで、建物には再入場しないでください。
* **部屋内に留まらざるを得ない場合の対処法：**
    * 落ち着いてください。
    * バスルームの洗面台に水を張ってください。
    * シーツやタオルを濡らし、ドアの隙間を塞いでください。
    * **111** へ電話し、部屋から避難できない旨をお伝えください。

### 危険箇所

* **上り傾斜の車道：** 当施設は傾斜地に位置しております。雨天・凍結時の運転には十分ご注意ください。
* **室内の階段** — お子様から目を離さないよう、十分にご注意ください。

# 基本情報・その他

### 虫について

* クイーンズタウン中心部にはサンドフライはいませんが、光に引き寄せられる小さな無害な虫が発生することがあります。換気のために窓を開けたい場合は、室内の照明を消すか、ブラインドを閉じた状態にしてください。
* ウェストコースト、グレノーキー／ダート川エリア、ミルフォード／ダウトフルサウンドへお出かけの際は、虫除けスプレーのご持参をお勧めします。

### 電源について

* ニュージーランドの電源コンセントは、一般的に 230/240 ボルト（50 Hz）です。お客様の電子機器を接続する際は、対応した定格であることをご確認ください。
